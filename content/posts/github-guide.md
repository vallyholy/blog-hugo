# GitHub 使用指南

GitHub 是一个全球领先的代码托管平台。

## 主要功能

1. **代码仓库** - 存储和管理代码
2. **版本控制** - Git 版本管理系统
3. **协作开发** - Pull Request, Issues
4. **GitHub Pages** - 免费静态网站托管

## 快速开始

```bash
git init
git add .
git commit -m "Initial commit"
git push
```

## GitHub Pages

GitHub Pages 可以免费托管静态网站，支持自定义域名。
